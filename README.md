# openedu-auto
Автоматический решатель тестов для OpenEdu

Тестировалось на Python 3.12

## Возможности
- Автоматическое решение через нейросеть
- Автоматическое решение с ответами из файла
- Сохранение ответов уже решённого курса в файл
- Авторизация через почту и через университет (УрФУ)

## Поддерживаемые типы заданий
- Одиночный и множественный выбор
- Поле ввода текста
- Выпадающий список
- Таблицы
- Кроссворд
